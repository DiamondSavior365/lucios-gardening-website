# Lucio's Gardening Brand Guide

## Brand Direction

Lucio's Gardening should feel premium, trustworthy, natural, family-owned, and professional. The website should take inspiration from luxury outdoor living websites while staying approachable for local homeowners.

## Brand Keywords

- Premium
- Clean
- Natural
- Reliable
- Family-owned
- Professional
- Outdoor living
- Garden care
- Landscape transformation
- Trustworthy

## Visual Style

The design should use large outdoor images, warm natural colors, clean typography, smooth spacing, and elegant call-to-action sections.

## Typography Direction

The site should use clean, elegant fonts. Headings should feel premium and confident. Body text should be readable and friendly.

Possible font direction:

- Serif or elegant display font for headings
- Clean sans-serif font for body text

For now, we can use default system fonts until we choose final Google Fonts.

## Button Style

Primary buttons should be:

- Rounded full
- Dark green background
- White text
- Uppercase or semi-bold text
- Smooth hover effect

Example button style:
`rounded-full bg-[#13291f] px-8 py-4 text-sm font-semibold uppercase tracking-[0.2em] text-white transition hover:bg-[#224735]`

## Layout Style

Sections should feel spacious and clean.

General section spacing:

- Mobile: `px-6 py-20`
- Desktop: `px-8 py-28`

General container width:

- `max-w-7xl mx-auto`

## Image Style

Images should feel high-end and natural.

Image guidelines:

- Use real project photos when possible
- Use large hero images
- Use rounded corners for cards
- Avoid low-quality blurry photos
- Show before-and-after transformations
- Prioritize clean lawns, trees, planters, gardens, and outdoor living areas

## Voice & Messaging

The writing should sound professional, confident, and approachable.

Example tone:
“Beautiful outdoor spaces, maintained with care.”

Avoid sounding too cheap or basic. The website should not only say “we mow lawns.” It should communicate that Lucio's Gardening improves the beauty, comfort, and value of outdoor spaces.

## Core Message

Lucio's Gardening provides reliable lawn care, tree care, landscaping, and outdoor maintenance with family pride, attention to detail, and a commitment to beautiful outdoor spaces.
